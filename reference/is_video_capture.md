# Test for a Python VideoCapture Object

This function tests whether an object is a Python VideoCapture object.

## Usage

``` r
is_video_capture(x)
```

## Arguments

- x:

  Any R object.

## Value

A logical indicating whether the object is a Python VideoCapture object
(TRUE) or not (FALSE).
