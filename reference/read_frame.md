# Read Specific Frame in a Python VideoCapture Object

This function reads a specific frame of a Python VideoCapture object and
returns it as Numpy array.

## Usage

``` r
read_frame(x, i)
```

## Arguments

- x:

  A Python VideoCapture object.

- i:

  The 1-indexed frame to be read.

## Value

A Numpy array.
