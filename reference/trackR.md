# App Launcher

This function starts an app launcher that gives the user access to all
the Shiny apps provided with trackRcv.

## Usage

``` r
trackR(...)
```

## Arguments

- ...:

  Parameters to be passed to
  [`shiny::runApp()`](https://rdrr.io/pkg/shiny/man/runApp.html).

## Value

This function does not return anything.

## See also

[`shiny::runApp()`](https://rdrr.io/pkg/shiny/man/runApp.html)
